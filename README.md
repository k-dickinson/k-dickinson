### `Primary Finding` &nbsp;·&nbsp; EvalTrust &nbsp;&nbsp;[repo →](https://github.com/k-dickinson/evaltrust)

A statistical auditor for AI model evals — it decides whether the gap between two evaluations is a **real improvement or just noise**, replacing raw score comparisons with significance testing.

- **McNemar**, paired **permutation**, and **equivalence tests (TOST)** · **power analysis** (minimum detectable effect) · **multiple-comparison correction** — all validated against SciPy & statsmodels
- Ships as a **CLI**, a **Python API**, and a **CI/CD GitHub Action** that gates merges on confidence intervals
- Adapters for Promptfoo and DeepEval result formats
